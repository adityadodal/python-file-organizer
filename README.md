# Python File Organizer

## Overview
This project is a Python script that automatically organizes files in a specified folder based on their file type. Files are categorized into folders like Images, Documents, Videos, and Others.

## Features
- Automatically detects files in the target folder
- Categorizes files by extension
- Creates folders dynamically if they don’t exist
- Safely ignores existing folders
- Moves uncategorized files to an "Others" folder

## Usage
1. Set the `TARGET_FOLDER` variable in `organizer.py` to the folder you want to organize.
2. Run the script:
```bash
python3 organizer.py
